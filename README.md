**Multi-Utility Toolkit**

A Python-based multi-functional console application demonstrating modular programming, package structuring, and usage of Python’s built-in modules.


**Overview**

This project implements a Menu-Driven Multi-Utility Toolkit that allows users to perform various daily life tasks such as:

Date and Time operations

Mathematical calculations

Random data generation

UUID creation

File handling

Module exploration using dir()

This project fulfills modular programming concepts by separating each feature into independent modules and integrating them under one program through a central main.py script.

**Features & Functionalities**

**1️⃣ Time & Date Operations (time_module.py)**

1. Display current date & time

2. Calculate difference between two dates

3. Format date in custom style

4. Stopwatch

5. Countdown timer

**Modules used:**
datetime, time


**2️⃣ Mathematical Operations (math_module.py)**

1. Factorial of a number

2. Compound Interest

3. Trigonometric values (sin, cos, tan)

4. Area of shapes (Circle, Rectangle, Triangle)

**Modules used:**
math


**3️⃣ Random Data Generation (random_module.py)**

1. Generate random integers

2. Generate random list

3. Secure random password

4. Generate OTP

**Modules used:**
random, string


**4️⃣ UUID Generator (uuid_module.py)**

1. Generate unique identifiers such as UUID4

**Modules used:**
uuid


**5️⃣ File Operations (file_module.py)**

1. Create a new text file

2. Write data to a file

3. Read file contents

4. Append data to a file

**Modules used:**
os


**6️⃣ Explore Module Attributes (explore_module.py)**

1. Shows available properties and functions of any module via dir()

**Modules used:**
built-in module: dir()


**Why This Project is Useful**

✅ Demonstrates Modular Programming Concept.

✅ Shows Working with Python Built-in Modules.

✅ Real-world examples of utilities like passwords, timers, etc.

✅ Clear code structure → easy to maintain & extend.
